Stick Hero Game:


This is a simple implementation of the Stick Hero Game using its basic mechanisms. 

How to play the game:
1. Run the code in java
2. To start playing, first left click once to start increasing the size of the stick and click again to stop.
3. Keep going unitl you lose the game
4. Once the game is over, you can either quit or restart from the buttons on the top right.
5. Additional feature is that you can restart the game anytime while playing using the button.
